Introduction

Hi There!

Our Work

Using a Spotify music dataset from Kaggle, we aim to provide insights on music and to predict popularity of songs using machine learning! The machine learning models used include SGD Regression, Ridge Regression, and LGBM Regression.

Our Motivation

By providing visualizations and a direct model that people can use for predicting song popularity, our product will help small artists understand what factors make songs popular. This can also improve the recommendation systems for music-sharing companies and other platforms!

Our Dataset

This dataset includes 50,000 rows and 18 columns. There are 10 different music genres including 'Electronic', 'Anime', 'Jazz', 'Alternative', 'Country', 'Rap', 'Blues', 'Rock', 'Classical', and 'Hip-Hop'. Other data types included are loudness, speechiness, danceability, instrumentalness, and more provided from Spotify.

Data Cleaning

Features that do not help in predicting song popularity, like identifiers and the date a data sample was obtained, were removed. Data samples with missing values were removed. Missing values in the "tempo" column were imputed with a value of 120.

Machine Learning

Ridge Regression

Ridge regression is a type of regularized linear regression that includes an L2 penalty. This has the effect of shrinking the coefficients for those input variables that do not contribute much to the prediction task.

  • R2 Score: 0.65364
  • Mean Absolute Error: 6.76776

Stochastic Gradient Descent

Stochastic Gradient Descent, or SGD, is an algorithm that can be used for machine learning. SGD works by computing gradients at random data points to greatly reduce the required memory for computations.

  • R2 Score: 0.65365
  • Mean Absolute Error: 6.76883

Light Gradient Boosting Machine

LightGBM is a gradient boosting framework based on decision trees to increase the efficiency of the model and reduces memory usage. LightGBM splits the tree leaf-wise as opposed to other boosting algorithms that grow tree level-wise.

  • R2 Score: 0.66935
  • Mean Absolute Error: 6.61593

Conclusion

What we found

From exploratory data analysis, it was found that the factors that make songs popular are not strongly correlated to one particular musical characteristic. However, the variable with the strongest influence on popularity was the music genre.

The three models made predictions on popularity with similar r-squared values and mean absolute errors. The model with the best performance was the LGBM Regressor. LGBM had the highest r-squared value and the lowest mean absolute error.

While the r-squared values have room for improvement, the overall performance is decent considering the task of predicting song popularity. Our regression models can be used to give a rough estimate for how popular a song will be. Try out the section below to see how popular your song might become!
